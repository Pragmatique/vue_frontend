# Frontent for common web application using Vue.
  Vuex was used for saving state.
  Vuetify was picked as layout framework.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
